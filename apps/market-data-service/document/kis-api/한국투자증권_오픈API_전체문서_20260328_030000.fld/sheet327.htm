개요
개요 장내채권 잔고조회 API입니다.

한국투자 HTS(eFriend Plus) &gt; [0979] 장내채권종합주문 화면의 "왼쪽 하단 잔고" 기능을 API로 개발한 사항으로, 해당 화면을 참고하시면 기능을 이해하기 쉽습니다.
