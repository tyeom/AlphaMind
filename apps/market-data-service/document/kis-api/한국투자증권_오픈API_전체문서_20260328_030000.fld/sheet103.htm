개요
개요 당사 대주가능 종목 API입니다.

한국투자 HTS(eFriend Plus) &gt; [0490] 당사 대주가능 종목 화면의 기능을 API로 개발한 사항으로, 해당 화면을 참고하시면 기능을 이해하기 쉽습니다.



※ 본 API는 다음조회가 불가합니다.
