개요
개요 NAV 비교추이(분) API입니다.

한국투자 HTS(eFriend Plus) &gt; [0244] ETF/ETN 비교추이(NAV/IIV) 좌측 화면 "분별" 비교추이 기능을 API로 개발한 사항으로, 해당 화면을 참고하시면 기능을 이해하기 쉽습니다.

실전계좌의 경우, 한 번의 호출에 최근 30건까지 확인 가능합니다.
