개요
개요 국내주식 예약주문 정정/취소 API 입니다.

*  정정주문은 취소주문에 비해 필수 입력값이 추가 됩니다.

   하단의 입력값을 참조하시기 바랍니다.



※ POST API의 경우 BODY값의 key값들을 대문자로 작성하셔야 합니다.

   (EX. "CANO" : "12345678", "ACNT_PRDT_CD": "01",...)
