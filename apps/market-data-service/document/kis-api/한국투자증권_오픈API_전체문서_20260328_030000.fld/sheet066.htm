개요
개요 ELW 민감도 추이(일별) API입니다.

한국투자 HTS(eFriend Plus) &gt; [0283] ELW 민감도 추이 화면의 "일자별" 민감도추이 기능을 API로 개발한 사항으로, 해당 화면을 참고하시면 기능을 이해하기 쉽습니다.
