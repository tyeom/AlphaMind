개요
개요 주식, ETF, ETN만 조회 가능하며 펀드는 조회 불가합니다.



​※ 55번 계좌(DC가입자계좌)의 경우 해당 API 이용이 불가합니다.

KIS Developers API의 경우 HTS ID에 반드시 연결되어있어야만 API 신청 및 앱정보 발급이 가능한 서비스로 개발되어서 실물계좌가 아닌 55번 계좌는 API 이용이 불가능한 점 양해 부탁드립니다.
