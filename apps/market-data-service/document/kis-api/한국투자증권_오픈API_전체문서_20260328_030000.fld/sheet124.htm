개요
개요 관심종목 그룹별 종목조회 API입니다.

① 관심종목 그룹조회 → ② 관심종목 그룹별 종목조회 → ③ 관심종목(멀티종목) 시세조회 순서대로 호출하셔서 관심종목 시세 조회 가능합니다.



※ 한 번의 호출에 최대 30종목의 시세 확인 가능합니다.



한국투자증권 Github 에서 관심종목 복수시세조회 파이썬 샘플코드를 참고하실 수 있습니다.
