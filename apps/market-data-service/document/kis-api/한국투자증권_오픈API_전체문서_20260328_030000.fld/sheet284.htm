개요
개요 해외주식 지연호가(아시아)의 경우 아시아 무료시세(지연호가)가 제공됩니다.



HTS(efriend Plus) [7781] 시세신청(실시간) 화면에서 유료 서비스 신청 시,

"해외주식 실시간호가 HDFSASP0" 을 이용하여 아시아국가 유료시세(실시간호가)를 받아보실 수 있습니다. (24.11.29 반영)



※ 지연시세 지연시간 : 홍콩, 베트남, 중국, 일본 - 15분지연
